## Education Sector

### Legal and Regulatory Requirements

UK educational institutions must comply with the following data protection legislation:

- **UK General Data Protection Regulation (UK GDPR)**
- **Data Protection Act 2018**

**Key Principles:**
- Use personal data fairly, lawfully, and transparently
- Use data for specified, explicit purposes
- Ensure data is adequate, relevant, and limited to what is necessary
- Keep data accurate and up to date
- Retain data only as long as necessary
- Protect data with appropriate security measures

**Special Categories:**
- Stronger legal protection applies to sensitive data (e.g., race, health, biometrics)

**Individual Rights:**
- Right to be informed, access, rectify, erase, restrict, and object to data processing
- Right to data portability
- Rights regarding automated decision-making and profiling
