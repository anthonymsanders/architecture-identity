# Microsoft Identity Management

## Overview
Microsoft provides a suite of identity management solutions for both on-premise and cloud environments, supporting education sector needs for secure access and compliance.

## Key Technologies
- Microsoft Active Directory (AD)
- Microsoft Entra ID (formerly Azure AD)
- Microsoft Identity Manager (MIM)
- ADFS (Active Directory Federation Services)

## Features
- Single Sign-On (SSO)
- Multi-Factor Authentication (MFA)
- Self-service password reset
- Integration with Microsoft 365 and other services
