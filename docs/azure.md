# Microsoft Azure Identity Management

## Overview
Microsoft Azure offers cloud-based identity management through Entra ID (formerly Azure AD), supporting hybrid and cloud-native architectures for education institutions.

## Key Technologies
- Microsoft Entra ID (Azure AD)
- Azure AD B2C (Business to Consumer)
- Azure AD Connect (hybrid sync)
- Conditional Access

## Features
- Cloud SSO and MFA
- Identity protection and risk-based access
- Integration with SaaS and on-premise apps
- Federation with other institutions
