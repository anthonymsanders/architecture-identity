# Microsoft and Third-Party: OpenText Identity Manager

## Overview
OpenText Identity Manager can be integrated with Microsoft identity solutions to provide advanced identity governance, lifecycle management, and compliance features.

## Key Technologies
- OpenText Identity Manager
- Microsoft Active Directory (AD)
- Microsoft Entra ID (Azure AD)

## Features
- Automated provisioning and deprovisioning
- Role-based access control
- Audit and compliance reporting
- Integration with cloud and on-premise systems
